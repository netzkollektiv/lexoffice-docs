.. _configuration:

Konfiguration
=============

Die Konfiguration des Modules befindet sich unter :menuselection:`Verkäufe --> lexoffice --> Konfiguration` 


Verbindung
--------------------------------

Im Tab *Verbindung* stellen Sie eine Verbindung zur lexoffice Applikation her und buchen darüber das Abonnement zur Verwendung der Applikation.

Sobald eine Verbindung besteht, werden Informationen zum lexoffice Account angezeigt.
Sobald ein Abonnement besteht und eine Verbindung herstellt ist, erscheint im Feld *Subscription Key* der Buchungscode.	
Ein bestehendes Abo kann über den Button "Kündigen" auf Wunsch gekündigt werden.

Gutschriften
--------------------------------

Hier können Sie einstellen, ob Gutschriften nach Erstellung automatisch nach lexoffice übernommen werden sollen, oder Sie die Übertragung lieber manuell vornehmen möchten.

Rechnungen
--------------------------------

Hier können Sie einstellen, ob Rechnungen nach Erstellung automatisch nach lexoffice übernommen werden sollen, oder Sie die Übertragung lieber manuell vornehmen möchten.

Kunden
--------------------------------

Wenn Sie einen Sammelkunden für alle Rechnungen verwenden möchten, können Sie dies hier einstellen. Wählen Sie "nein", um für jeden Magento Kunden automatisch auch einen Kunden in Lexoffice anzulegen.
